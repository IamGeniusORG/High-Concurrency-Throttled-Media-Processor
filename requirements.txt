==================================================
PROJECT REQUIREMENTS & INSTALLATION GUIDE
==================================================

To run both parts of this project perfectly (the custom Go application and the yt-dlp integration), you must have the following three programs installed on your computer.

--------------------------------------------------
1. REQUIRED SOFTWARE OVERVIEW
--------------------------------------------------
1. Go (Golang)
   - Why: Required to compile and run the High-Concurrency Go Application (`main.go`).
   - Official Website: https://go.dev/dl/

2. yt-dlp
   - Why: Required to download from complex, protected streaming sites like YouTube.
   - Official Website: https://github.com/yt-dlp/yt-dlp#installation

3. FFmpeg
   - Why: Required by yt-dlp to merge high-definition audio and video streams into native .mp4 containers.
   - Official Website: https://ffmpeg.org/download.html

--------------------------------------------------
2. AUTOMATIC INSTALLATION (RECOMMENDED)
--------------------------------------------------
Copy and paste the commands below into your terminal based on your operating system:

[WINDOWS] (Open PowerShell as Administrator)
winget install GoLang.Go
winget install yt-dlp.yt-dlp
winget install Gyan.FFmpeg

[macOS] (Open Terminal with Homebrew installed)
brew install go
brew install yt-dlp
brew install ffmpeg

[LINUX (Debian/Ubuntu)] (Open Terminal)
sudo apt update
sudo apt install golang ffmpeg
sudo wget https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -O /usr/local/bin/yt-dlp
sudo chmod a+rx /usr/local/bin/yt-dlp

--------------------------------------------------
3. HOW TO VERIFY YOUR INSTALLATION
--------------------------------------------------
After installing, close your terminal, open a fresh one, and run these three commands to verify everything is perfectly set up:

go version
yt-dlp --version
ffmpeg -version

If all three print out a version number, you are ready to start downloading!
