PROJECT REQUIREMENTS

To run both parts of this project perfectly (the custom Go application and the yt-dlp integration), you must have the following three programs installed on your computer:

1. Go (Golang) - Required to compile and run the High-Concurrency Go Application (main.go).
2. yt-dlp - Required to download from complex, protected streaming sites like YouTube.
3. FFmpeg - Required by yt-dlp to merge high-definition audio and video streams into native .mp4 containers.

--------------------------------------------------
INSTALLATION INSTRUCTIONS
--------------------------------------------------

[WINDOWS]
Open PowerShell and run the following commands to install everything via Winget:
1. winget install GoLang.Go
2. winget install yt-dlp.yt-dlp
3. winget install Gyan.FFmpeg

[macOS]
Open the Terminal and run the following commands to install everything via Homebrew:
1. brew install go
2. brew install yt-dlp
3. brew install ffmpeg

[LINUX (Debian/Ubuntu)]
Open the Terminal and run the following commands:
1. sudo apt update
2. sudo apt install golang ffmpeg
3. sudo wget https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -O /usr/local/bin/yt-dlp
4. sudo chmod a+rx /usr/local/bin/yt-dlp
